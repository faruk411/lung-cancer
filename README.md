## Veri Seti Linki

https://www.kaggle.com/datasets/rm1000/lung-cancer-histopathological-images

## Proje Detayı 

Veri setimde 25 bin resim bulunmakta, 5 adet sınıf içermektedir. bu sınıflar şunlardır => kolon adenokarsinomu - İyi huylu kolon dokusu - Akciğer adenokarsinomu - Akciğer skuamöz hücreli karsinomu - İyi huylu akciğer dokusu
Oluşturduğum model bu sınıflara bağlı olarak tahminde bulunur. 
Modelimin doğruluk oranı 93% ' tür.
